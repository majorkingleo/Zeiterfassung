Abwesenheitszeiten Buchen

Hier können sie ein und die selbe Buchung über mehrer Tage hinweg durchführen. ZB: 14 Urlaub buchen.

Sie können eine Start und einen Endzeit selbst festlegen, oder den gesammten Tag auswählen. Haben sie den gesammten Tag ausgwählt so wird automatisch aufgrund Ihrere Sollarbeitszeit eine Buchungszeile eingeführt. Also bei einer 38,5 Stundenwoche werden automatisch 7,7 Stunden pro Tag gebucht.

An Samstagen, Sonn und Feiertagen werden automatisch keine Buchungen durchgeführt.

Sollten Sie bereits eine Buchungszeile an einem in Frage kommen Tag eingetragen haben, so wird überprüft, ob sich die Buchungen überschneiden. Sollte dies der Fall sein, so wird an diesem Tag keine Buchung durchgeführt.

Sie können nachträglich alle Buchungen bearbeiten, und auch wieder löschen.
