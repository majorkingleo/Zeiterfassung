Versionsänderungsprotokoll

Dieses Protokoll listet die Änderungen von Version zu Version im Programm auf.

Version 0.9.0 (2010-09-15 bis 2010-09-20)

  • Einbau des Menüs Abwesenheitszeiten Buchen. Damit können nun Buchungen über mehrere Tage hinweg einfach durchgeführt werden.
  • Hilfemenüs über die F1 Taste zugänglich gemacht.

Version 0.8.3 (2010-07-27 bis 2010-08-11)

  • Verbesserung des Update Managers.
  • Unterstützung für 64Bit Java mit allen Biblotheken.
  • Verbesserung der Proxy Unterstützung.

Version 0.8.2 (2010-07-24 bis 27)

  • Besseres Speichern der Tabellenbreiten für den aktuellen Buchungstag.
  • Verbesserung des Proxy Support.
  • Hinzufügen eines Update Managers.

Version 0.8.1 (2010-07-22)

  • Bugfix beim Blättern des Monatsberichts über den Jänner hinweg.

Version 0.8 (2010-07-20)

  • Montsreport der über die Projektaktivitäten pro Monat berichtet hinzugefügt.

Version 0.7.1 (2010-06-22)

  • Bugfixes

Version 0.7 (2010-06-20)

  • Datenbank Import und Export hinzugefügt.
  • Performance Verbesserungen
  • Es wird nun immer das LookAndFeel System automatisch beim ersten mal Starten aktiviert.
  • Es gibt nun eine Demo Version mit bereist vorbereiteten Projektdaten
  • Es gibt nun eine Single User Variante

Version 0.6.5 (2010-05-29)

  • Mittels des Parameters "UseCustomersAndProjects" kann man nun die Kunden und Projekte deaktivieren.

Version 0.6.4 (2010-05-13)

  • Neuer Wizard "Benutzer anlegen"

Version 0.6.3 (2010-05-12)

  • Automatisch beim ersten Starten die länderspezifischen Feiertage vorbelegen.


Version 0.6.2 (2010-05-08)

  • Diverse Bugfixes
  • Das Datenbankpasswort wird nun verschlüsselt abgespeichert


Version 0.6.1 (2010-05-08)

  • Verbesserungen am Einrichtungs-Assistenten
  • Dieverse Bugfixes und Verbesserungen


Version 0.6 (2010-05-06)

  • Bugfix beim Anzeigen des Gleitzeitkontos. Hier wurden bei der Berechnung die Feiertage bei der Sollarbeitszeitberechnung nicht berücksichtigt.

Version 0.5.9 (2010-05-03)

  • Neuer, lokaler Parameter "LookAndFeel", der das Aussehen der Benutzeroberfläche beeinflusst.

Version 0.5.8 (2010-03-23)

  • Bugfix beim Berechnen der Resturlaubszeit, wenn im aktuellen Monat kein Urlaub gebucht wurde.

Version 0.5.7 (2010-03-14)

  • Desktop Icon erstellen Funktion eingebaut.

Version 0.5.6 (2010-02-21 bis 2010-03-01)

  • Bei den Reports werden nun auch österreichische Monatsnamen verwendet.
  • Dieverse Bugfixes

Version 0.5.5 (2010-02-20)

  • Splash Screen eingebaut.

Version 0.5.4 (2010-01-17)

  • Österreichische Monatsnamen eingebaut.

Version 0.5.3 (2009-10-13)

  • Bugfixes im Usermanagment

Version 0.5.2 (2009-10-07)

  • Anpassungen für Oracle.

Version 0.5.1 (2009-10-03)

  • Autologin Eingebaut.

Version 0.5.0 (2009-09-23)

  • Kunde, Projekte und Unterprojekte können nun angelegt werden und es ist möglich Zeit darauf zu buchen.
  • Wenn gewünscht, so werden die Unterprojekte auch gleich automatisch angelegt.

Version 0.4.2 (2009-08-27)

  • Autovervollständigung verbessert. Es werden nun sinnvolle Zeiten, sowie bereits verwendete Kommentare vorgeschlagen.

Version 0.4.1 (2009-08-26)

  • Tabellenspaltenbreite verbessert.
  • Mit drücken der Escape Taste, werden nun alle Fenster automatisch geschlossen.
  • Autovervollständigung für normale Textfelder in einer Tabelle hinzugefügt.

Version 0.4.0 (2009-08-24)

  • Arbeitszeit pro Woche in den Monatseinstellungen darf nun auch 0 sein.
  • Überstundenschema "KEINES", "PAUSCHALIST" und "ÜBERSTUNDENSCHEMA_01" eingebaut.
  • Sollstunden werden nun im Format HH:mm angezeigt.
  • Die Minimale Anzahl ein einzgebenenden Zeichen im Kommentar bei den Zeiteinträgen ist nun über den Parameter NumberOfMinimumCommentChars konfigurierbar
  • Beim Monatsbericht einen Schließen Button hinzugefügt

Version 0.3.2 (2009-08-23)

  • Die Bilder in den Hilfedateien werden wieder angezeigt
  • Dieses Protokoll wurde eingebaut.
  • Geht eine der Monatsberechnung schief, so sieht der Benutzer nun ein kleines Info Icon und wenn er darauf klickt, werden die Fehlermeldungen inklusive Lösungsvorschläge angezeigt.
