Editierbarkeit festlegen

Hier können Sie Festlegen ab welchem Tag die jeweiligen Benutzer Ihre Zeiten editieren dürfen. Der Gedanke dabei ist das nicht festzulegen ab wann ein Monat nicht mehr editiert werden darf, sondern genau umgekehrt. Ab welchem Zeitpunkt die Tage editiert werden sein dürfen.

Zum einfacheren festlegen eines neuen Monatsabschlußtermins für alle Benutzer kann einfach ein neuer Termin für in das Eingabefeld eingetragen werden und mit dem Button Festlegen wird das Datum allen Benutzern zugewiesen.

Das Eingabefeld ist automatisch dem dem aktuellen Tagesdatum vorbelegt.

Neue Benutzer müssen explizit hier in diesem Dialog eingetragen werden, damit sie Zeiteinträge erstellen können.
